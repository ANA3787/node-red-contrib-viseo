# Node-RED

A Node-RED node to add AI to your chatbot.
You can talk to IRMA (Image Recognition Magic App) [here](irma.viseo.io) to discover its functionalities. 

This node is part of project [node-red-contrib-viseo](https://github.com/NGRP/node-red-contrib-viseo) powered by [VISEO](http://www.viseo.com) Technologies.

## Quick Start

```
npm install node-red-contrib-viseo-irma
```
